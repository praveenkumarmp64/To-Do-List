# 📝 To-Do List Web Application

A simple and responsive To-Do List web application built using **HTML, CSS, and JavaScript** with **LocalStorage** support.

---

## 🚀 Features

- User Login System
- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Completed
- Separate Pending & Completed Tasks
- Data stored in Browser LocalStorage
- Simple Responsive UI
- Logout Functionality
